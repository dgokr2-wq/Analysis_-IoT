# AI4I Predictive Maintenance Analysis

Данный репозиторий содержит материалы индивидуального проекта по дисциплине **«Интеллектуальный анализ больших данных»**. Проект посвящен анализу данных промышленного оборудования на основе датасета **AI4I 2020 Predictive Maintenance Dataset** и включает реализацию аналитической части на **Python** и визуализацию в **R**.

В рамках проекта выполнены загрузка и предобработка данных, разведочный анализ, проверка статистических гипотез, выявление аномальных режимов работы оборудования, построение модели прогнозирования отказов и визуализация результатов. Python-часть используется для основной аналитики, моделирования и работы интерактивного приложения, а R-часть — для построения ключевых графиков с использованием библиотеки **ggplot2**.

## Состав репозитория

- `ai4i_predictive_maintenance_app.py` — основное приложение и аналитика на Python
- `ai4i_visualization.R` — построение ключевых графиков в R
- `ai4i2020.csv` — исходный датасет
- `requirements.txt` — зависимости для Python
- `README.md` — описание проекта

## Что реализовано в проекте

В проекте выполнены следующие этапы:
- анализ структуры датасета;
- проверка пропусков и дубликатов;
- создание новых признаков;
- построение описательной статистики;
- визуализация распределений и сравнений;
- проверка статистических гипотез;
- выявление аномалий;
- построение модели прогнозирования отказов;
- анализ важности признаков;
- подготовка графиков для отчета.

## Используемые технологии

Для реализации проекта использовались:
- **Python**: pandas, numpy, matplotlib, scipy, scikit-learn, tkinter
- **R**: readr, dplyr, ggplot2
